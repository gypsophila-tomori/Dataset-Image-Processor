# Dataset Image Processor

一个用于数据集图片预处理的 GUI 工具，支持批量筛选、缩放、旋转和重命名图片。

## 功能特性

- 🖼️ 缩略图网格浏览
- ✅ 逐张筛选（保留/跳过）
- 🔄 图片旋转（90°、180°、270°）
- 📏 等比例缩放
- 🏷️ 批量重命名（连续序号）
- ⚡ 懒加载和缓存优化
- 📊 处理进度显示
- 📝 自动生成处理日志

## 安装

```bash
pip install -r requirements.txt
```

## 使用方法

1. 运行程序：
```bash
python main.py
```

2. 选择输入文件夹（包含原始图片）
3. 选择输出文件夹（保存处理后的图片）
4. 浏览图片，标记保留/跳过
5. 设置处理参数（缩放比例、文件命名等）
6. 点击"开始批量处理"

## 快捷键

- `←/→` - 上一张/下一张
- `Space` - 切换保留/跳过
- `Ctrl+O` - 选择输入文件夹
- `Ctrl+S` - 开始处理

## 输出文件

- 处理后的图片：`train_00001.png`, `train_00002.png`, ...
- `processed_log.txt` - 处理日志
- `skipped_files.txt` - 跳过的文件列表

## 系统要求

- Python 3.8+
- PyQt5
- Pillow
